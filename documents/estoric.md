# If statements
`1I"Yeah"]"Nah"]`
<code>
1I"Yeah"]"Nah"]
1                Push 1
 I      ]     ]  If statement
  "Yeah"         If true, push "Yeah"
         "Nah"   If false, push "Nah"
</code>
